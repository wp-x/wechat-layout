# 文章类型预设

预设决定组件搭配倾向与节奏，主题色可任意组合。

## 干货教程型

适用：方法论、教程、清单、工具推荐。
- 结构：导语卡 → 编号分隔符分节 → 每节"二级标题 + 正文 + 列表卡片/步骤时间线" → 结尾重点卡片。
- 多用：步骤时间线、列表卡片、提示条、对比双栏、代码块。
- 节奏：段落短，每 3-4 段一个视觉组件。

## 观点评论型

适用：热点评论、行业观察、个人思考。
- 结构：首字下沉开篇 → 编号分隔符分节 → 金句卡穿插 → 居中强调收尾。
- 多用：金句卡、高亮标注、引用块、数字亮点。
- 节奏：以文字为主，组件克制，突出观点句。

## 情感故事型

适用：情感、人物、随笔。
- 结构：首字下沉 → 简单分隔线换场 → 金句卡点题 → 引导关注卡。
- 多用：简单分隔线（代替编号分隔符）、金句卡、居中强调。
- 少用：卡片类组件，避免打断叙事；高亮仅用于点题句。

## 资讯速报型

适用：新闻汇总、周报、动态盘点。
- 结构：导语卡 → 每条资讯"二级标题 + 简短正文" → 无需总结卡。
- 多用：二级标题区块、数字亮点、嵌套说明（标注来源）。
- 节奏：密集紧凑，段落间距可减为 16px。

## 商务通告型

适用：企业公告、活动预告、招聘、产品发布。
- 结构：重点卡片开门见山 → 信息卡片列关键信息（时间/地点/方式）→ FAQ → 引导关注卡。
- 多用：信息卡片、FAQ、提示条、数字亮点。
- 推荐主题：商务黑金、科技蓝、中国红。

## 深度长文型

适用：万字长文、深度报道、研究分析。
- 结构：导语卡 → 一级标题区块分大章 → 编号分隔符分小节 → 引用块标注来源 → 结尾重点卡片。
- 多用：一级标题区块、引用块、数字亮点、对比双栏。
- 推荐主题：极简黑白、商务黑金。
- 每 800 字左右安排一个视觉组件或图片占位，防止阅读疲劳。
