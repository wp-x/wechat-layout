# 配色主题库

每套主题提供统一令牌，替换 `components.md` 模板中的 `{{令牌}}` 即可。
`{{阴影}}` 用于 box-shadow 的 rgba 颜色；`{{阴影}}` 后的数值（0.1/0.15/0.25）保持组件模板原样。

## 令牌说明

| 令牌 | 用途 |
|---|---|
| `{{主题色}}` | 标题边框、分隔符、首字、强调文字 |
| `{{文字主色}}` | 正文颜色 |
| `{{标题色}}` | 标题文字（比正文更深） |
| `{{高亮背景}}` | 重点标注底色 |
| `{{高亮文字}}` | 重点标注文字色 |
| `{{辅助底色}}` | 卡片背景（极浅色） |
| `{{卡片文字}}` | 卡片内正文色 |
| `{{次要文字}}` | 说明、注释、来源 |
| `{{分割线}}` | 细线颜色 |
| `{{阴影}}` | box-shadow 的 RGB 值 |

## 自定义主题

用户给出品牌色时，按以下方法派生整套令牌：
- 主题色 = 品牌色
- 高亮文字/卡片文字 = 品牌色加深一档（HSL 亮度 -15%）
- 高亮背景 = 品牌色淡化至 90% 亮度
- 辅助底色 = 品牌色淡化至 97% 亮度
- 阴影 = 品牌色的 RGB 值
- 其余令牌沿用清新绿的中性灰
